# MNIST Binary Classification (2 vs 7) - CNN

This project implements a Convolutional Neural Network (CNN) to distinguish between digits 2 and 7 from the MNIST dataset.

## Project Structure
* `1_Modele.py`: Script to train the CNN and save weights.
* `2_Evaluation.py`: Script to evaluate the model and generate a confusion matrix.
* `Model.hdf5`: Pre-trained weights for the classifier.
* `INF5081_TP2_RAPPORT.pdf`: Detailed project report.

## How to Run
1. Install dependencies: `pip install tensorflow numpy matplotlib scikit-learn`
2. Run evaluation: `python 2_Evaluation.py`
